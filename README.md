# OpenPowerlifting Preprocessing
Some data preprocessing conducted on a snapshot of the [OpenPowerlifting](https://www.kaggle.com/open-powerlifting/powerlifting-database) dataset for a small regression project.<br>
Data snapshot available here: [powerlifting-database](https://www.kaggle.com/datasets/dansbecker/powerlifting-database)<br>
**Dependencies:** `numpy`, `pandas`<br>
